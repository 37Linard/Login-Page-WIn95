## 🚀 Quick Start

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/37Linard/Login-Page-WIn95.git
cd Login-Page-WIn95
```

### 2️⃣ Abra no navegador

Simplesmente clique em `index.html` ou use um servidor local:

```bash
# Com Python 3
python -m http.server 8000

# Com Node.js
npx http-server
```

Depois acesse `http://localhost:8000`

## 🎨 Design

O design segue fielmente a estética do **Windows 95**:

- Bordas biseladas em cinza
- Botões com efeito 3D pressionado/solto
- Barra de título com ícone e botões de controle
- Caixa de diálogo clássica
- Tipografia bitmap (Trebuchet MS)
