# 🪟 Login Win95

Uma página de login retrô inspirada no clássico Windows 95. Ideal pra quem curte nostalgia com um toque de HTML, CSS e JS raiz.  

![preview](preview.png) <!-- imagem opcional, caso tenha um print do projeto -->

---

## 🧠 Sobre o Projeto

Este projeto foi criado com o objetivo de praticar desenvolvimento front-end e experimentar um design retrô com cara de sistema operacional antigo (Windows 95 vibes). É uma landing page de login que simula janelas, botões e caixas de diálogo com o visual old-school.

---

## ⚙️ Tecnologias Usadas

- HTML5
- CSS3
- JavaScript (puro, na moral)
- Estética retrô manual (sem Bootstrap, Tailwind ou frameworks moderninhos)

---

## 💾 Como Rodar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/37Linard/Login-Page-WIn95
````

2. Abra o arquivo `index.html` no navegador.
